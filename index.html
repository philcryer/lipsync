lipsync by phil.cryer

A lightweight service that provides command-line, Dropbox like syncing

lipsync is an open source, lightweight service that provides command-line, Dropbox - like syncing in Linux using OpenSSH, rsync, unison and lsyncd. lipsync is a the realization of a popular blog post of mine named HOWTO build your own open source Dropbox clone. Since posting I've recieved a great deal of interest, and had time to test and architect a workable solution. Thanks to everyone that read, commented and encourged the further development of this idea!

Dependencies

Server - Linux 2.6.x, rsync, unison, opensshd
Client - Linux 2.6.x, rsync, unison, openssh, lsyncd

Caveats

Much of my initial feedback asked for a graphical user interface and support for Windows. Please know that I have attempted neither, instead deciding to focus on the server-side, fleshing out all the options and performance. Since most of the software used for lipsync is cross-platform, I'd be interested in working with others that want to build a gui front end to talk to the backend. I've been interested in the work done on the Sparkleshare project, and since they use a few different backends I hope that the software is agnostic enough to adapt to use what I've created.
